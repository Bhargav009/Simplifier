Simplifier makes creation of forms easy and provides the analytics of the data.

Technical Stack:
1. Angular 7
2. Google firebase (Realtime database and Authentication)
3. Bootstrap
4. Form.io
5. Google Charts Api
6. Ag-Grid
7. Ngx-Bootstrap

Features:
1. Supports Google, Github and Email sigin.
2. Clean and user-friendly UI.
3. Analytics.
4. Ag-Grid.
